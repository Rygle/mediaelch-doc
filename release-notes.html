Release Notes / Changelog¶

Note

This information is drawn from the GitHub MediaElch project changelog:
https://github.com/Komet/MediaElch/blob/master/changelog.md

Next Release (tbd)¶

The next release will affect your scraper settings. You’ll therefore have to reconfigure your scraper settings. You may also need to delete MediaElch.sqlite. See: https://mediaelch.github.io/mediaelch-doc/faq.html#where-are-mediaelchs-settings-stored
